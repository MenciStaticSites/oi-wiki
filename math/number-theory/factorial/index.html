    阶乘取模

    引入

    本文讨论了某一模数下阶乘计算的相关结论，并提供一种时间复杂度线性相关于模数大小的计算方法，因而该方法主要适用于模数不太大（∼106∼106）的情形。除了本文介绍的方法外，根据场景不同，还可以应用 多项式技术 进行快速计算。

    根据 中国剩余定理，阶乘取模问题可以转化为模数为素数幂 𝑝𝛼pα 的情形。在处理这类问题时，常常需要对于素数 𝑝p 和正整数 𝑛n，将阶乘 𝑛!n! 中的所有因子 𝑝p 都提取出来，进而得到分解：

    𝑛!=𝑝𝜈𝑝(𝑛!)(𝑛!)𝑝.n!=pνp(n!)(n!)p.

    其中，𝜈𝑝(𝑛!)νp(n!) 表示阶乘 𝑛!n! 的素因数分解中 𝑝p 的幂次，(𝑛!)𝑝(n!)p 表示在阶乘 𝑛!n! 的结果中去除所有 𝑝p 的幂次得到的整数。本文将讨论 (𝑛!)𝑝(n!)p 在素数（幂）模下的余数以及幂次 𝜈𝑝(𝑛!)νp(n!) 的具体计算方法。

    这种分解在解决阶乘同时出现在所求表达式的分子和分母的问题时尤为有用，比如 计算某一模数下的二项式系数。对于这类问题，分子和分母中 𝑝p 的幂次可以直接相减，而与 𝑝p 互素的部分 (𝑛!)𝑝(n!)p 则可以利用 乘法逆元 计算。

    本文还介绍了与上述问题相关的 Wilson 定理及其推广、Legendre 公式和 Kummer 定理等内容。

    Wilson 定理

    Wilson 定理给出了判断某个自然数是素数的一个充分必要条件。

    Wilson 定理

    对于自然数 𝑛 >1n>1，当且仅当 𝑛n 是素数时，(𝑛 −1)! ≡ −1(mod𝑛)(n−1)!≡−1(modn)。

    证明

    首先，证明对于素数 𝑝p 有 (𝑝 −1)! ≡ −1(mod𝑝)(p−1)!≡−1(modp)。对于这一点，可以利用 同余方程 或 原根 得到两种简洁的证明，此处略去不表。下面提供前置知识较少的一种证明方法：

    当 𝑝 =2p=2 时，命题显然成立。下面设 𝑝 ≥3p≥3，继而要证明 𝐙𝑝Zp 中所有非零元素（即同余类）的积为 ―――−1−1―。因为 𝐙𝑝Zp 中所有非零元素 ――𝑎a― 都有逆元 ――𝑎−1a―−1，于是 𝐙𝑝Zp 中彼此互逆的元素乘积为 ――11―。但是要注意 ――𝑎a― 和 ――𝑎−1a―−1 可能相等：――𝑎 =――𝑎−1a―=a―−1，当且仅当 𝑎2 ≡1(mod𝑝)a2≡1(modp)，即

    0≡𝑎2−1≡(𝑎+1)(𝑎−1),(mod𝑝)0≡a2−1≡(a+1)(a−1),(modp)

    从而，𝑎 ≡1(mod𝑝)a≡1(modp) 或 𝑎 ≡ −1(mod𝑝)a≡−1(modp)。这说明 𝐙𝑝 ∖{――0,――1,―――−1}Zp∖{0―,1―,−1―} 中所有元素的乘积为 ――11―，进而 𝐙𝑝Zp 中所有非零元素的积为 ―――−1−1―。

    反过来，对于合数 𝑛n 的情形，要证明 (𝑛 −1)! ≢ −1(mod𝑛)(n−1)!≢−1(modn)。利用反证法，不妨设 (𝑛 −1)! ≡ −1(mod𝑛)(n−1)!≡−1(modn)，亦即存在整数 𝑘k 使得 (𝑛 −1)! =𝑘𝑛 −1(n−1)!=kn−1 成立。因为 𝑛n 是合数，必然存在素数 𝑝 <𝑛p<n 使得 𝑛 =𝑝𝑚n=pm，所以 (𝑛 −1)! =𝑘𝑝𝑚 −1 ≡ −1(mod𝑝)(n−1)!=kpm−1≡−1(modp)。但是，乘积 (𝑛 −1)!(n−1)! 中必然已经出现 𝑝p，故而一定有 (𝑛 −1)! ≡0(mod𝑝)(n−1)!≡0(modp)。这一矛盾就说明了 (𝑛 −1)! ≢ −1(mod𝑛)(n−1)!≢−1(modn)。

    利用本文的记号，Wilson 定理可以写作 (𝑝!)𝑝 ≡ −1(mod𝑝)(p!)p≡−1(modp)。

    推广

    Wilson 定理可以推广到一般模数的情形。

    定理（Gauss）

    对于自然数 𝑚 >1m>1，有

    ∏1≤𝑘<𝑚, 𝑘⟂𝑚𝑘≡±1(mod𝑚).∏1≤k<m, k⟂mk≡±1(modm).

    而且，余数中的 ±1±1 取值为 −1−1 当且仅当模 𝑚m 的 原根存在，即 𝑚 =2,4,𝑝𝛼,2𝑝𝛼m=2,4,pα,2pα 时，其中 𝑝p 是奇素数且 𝛼α 是正整数。

    证明

    // Calculate (n!)_p mod p.
int factmod(int n, int p) {
  // Pretreatment.
  std::vector<int> f(p);
  f[0] = 1;
  for (int i = 1; i < p; ++i) {
    f[i] = (long long)f[i - 1] * i % p;
  }
  // Recursion.
  int res = 1;
  while (n > 1) {
    if ((n / p) & 1) res = p - res;
    res = (long long)res * f[n % p] % p;
    n /= p;
  }
  return res;
}

    // Calculate (n!)_p mod pa.
int factmod(int n, int p, int pa) {
  // Pretreatment.
  std::vector<int> f(pa);
  f[0] = 1;
  for (int i = 1; i < pa; ++i) {
    f[i] = i % p ? (long long)f[i - 1] * i % pa : f[i - 1];
  }
  // Recursion.
  bool neg = p != 2 || pa <= 4;
  int res = 1;
  while (n > 1) {
    if ((n / pa) & neg) res = pa - res;
    res = (long long)res * f[n % pa] % pa;
    n /= p;
  }
  return res;
}

    预处理的时间复杂度为 𝑂(𝑝𝛼)O(pα)，单次询问的时间复杂度为 𝑂(log𝑝⁡𝑛)O(logp⁡n)。

    幂次的计算

    本节讨论阶乘 𝑛!n! 中 𝑝p 的幂次 𝜈𝑝(𝑛!)νp(n!) 的计算，它可以用于计算二项式系数的余数。因为二项式系数中，分子和分母都会出现阶乘，而分子和分母中素数 𝑝p 能否互相抵消，就成为了决定最后的余数的重要因素。

    Legendre 公式

    阶乘 𝑛!n! 中素数 𝑝p 的幂次可以通过 Legendre 公式计算，而且与 𝑛n 在 𝑝p 进制下的表示有关。

    Legendre 公式

    对于正整数 𝑛n，阶乘 𝑛!n! 中含有的素数 𝑝p 的幂次 𝜈𝑝(𝑛!)νp(n!) 为

    𝜈𝑝(𝑛!)=∞∑𝑖=1⌊𝑛𝑝𝑖⌋=𝑛−𝑆𝑝(𝑛)𝑝−1,νp(n!)=∑i=1∞⌊npi⌋=n−Sp(n)p−1,

    其中，𝑆𝑝(𝑛)Sp(n) 为 𝑝p 进制下 𝑛n 的各个数位的和。特别地，阶乘中 22 的幂次是 𝜈2(𝑛!) =𝑛 −𝑆2(𝑛)ν2(n!)=n−S2(n)。

    证明

    因为

    𝑛!=1×2×⋯×𝑝×⋯×2𝑝×⋯×⌊𝑛/𝑝⌋𝑝×⋯×𝑛.n!=1×2×⋯×p×⋯×2p×⋯×⌊n/p⌋p×⋯×n.

    其中，𝑝p 的倍数的乘积为 𝑝 ×2𝑝 ×⋯ ×⌊𝑛/𝑝⌋𝑝 =𝑝⌊𝑛/𝑝⌋⌊𝑛/𝑝⌋!p×2p×⋯×⌊n/p⌋p=p⌊n/p⌋⌊n/p⌋!，而 ⌊𝑛/𝑝⌋!⌊n/p⌋! 可能继续出现 𝑝p 的倍数。所以，对于幂次，有递推关系：

    𝜈𝑝(𝑛!)=⌊𝑛/𝑝⌋+𝜈𝑝(⌊𝑛/𝑝⌋!).νp(n!)=⌊n/p⌋+νp(⌊n/p⌋!).

    将它展开就得到 Legendre 公式。

    要证明第二个等号，首先将 𝑛n 展开为 𝑝p 进制，这相当于将它写作如下和式：

    𝑛=𝑛ℓ𝑝ℓ+⋯+𝑛1𝑝+𝑛0=ℓ∑𝑘=0𝑛𝑘𝑝𝑘.n=nℓpℓ+⋯+n1p+n0=∑k=0ℓnkpk.

    // Obtain multiplicity of p in n!.
int multiplicity_factorial(int n, int p) {
  int count = 0;
  do {
    n /= p;
    count += n;
  } while (n);
  return count;
}

    Kummer 定理

    组合数对一个数取模的结果，往往构成分形结构，例如谢尔宾斯基三角形就可以通过组合数模 22 得到。

    如果仔细分析，𝑝p 是否整除组合数其实和上下标在 𝑝p 进制下减法是否需要借位有关。这就有了 Kummer 定理。

    Kummer 定理

    素数 𝑝p 在组合数 (𝑚𝑛)(mn) 中的幂次，恰好是 𝑝p 进制下 𝑚m 减掉 𝑛n 需要借位的次数，亦即

    𝜈𝑝((𝑚𝑛))=𝑆𝑝(𝑛)+𝑆𝑝(𝑚−𝑛)−𝑆𝑝(𝑚)𝑝−1.νp((mn))=Sp(n)+Sp(m−n)−Sp(m)p−1.

    特别地，组合数中 22 的幂次是 𝜈2((𝑚𝑛)) =𝑆2(𝑛) +𝑆2(𝑚 −𝑛) −𝑆2(𝑚)ν2((mn))=S2(n)+S2(m−n)−S2(m).

    证明

    首先证明下面的表达式。为此，利用 Legendre 公式，有

    𝜈𝑝((𝑚𝑛))=𝜈𝑝(𝑚!)−𝜈𝑝(𝑛!)−𝜈𝑝((𝑚−𝑛)!)=∞∑𝑖=1(⌊𝑚𝑝𝑖⌋−⌊𝑛𝑝𝑖⌋−⌊𝑚−𝑛𝑝𝑖⌋)=𝑆𝑝(𝑛)+𝑆𝑝(𝑚−𝑛)−𝑆𝑝(𝑚)𝑝−1.νp((mn))=νp(m!)−νp(n!)−νp((m−n)!)=∑i=1∞(⌊mpi⌋−⌊npi⌋−⌊m−npi⌋)=Sp(n)+Sp(m−n)−Sp(m)p−1.

    该表达式可以理解为 𝑝p 进制下 𝑚m 减掉 𝑛n 需要借位的次数。因为如果在计算第 𝑖i 位（最低位下标是 11）时存在不够减需要借位的情况，那么相减的结果中第 𝑖i 位之前的数字 ⌊𝑚−𝑛𝑝𝑖⌋⌊m−npi⌋，其实是 𝑚m 中第 𝑖i 位之前的数字 ⌊𝑚𝑝𝑖⌋⌊mpi⌋，减去一（即借掉的一），再减去 𝑛n 中第 𝑖i 位之前的数字得到的差值 ⌊𝑛𝑝𝑖⌋⌊npi⌋，所以，差值

    ⌊𝑚𝑝𝑖⌋−⌊𝑛𝑝𝑖⌋−⌊𝑚−𝑛𝑝𝑖⌋=1⌊mpi⌋−⌊npi⌋−⌊m−npi⌋=1

    当且仅当发生了一次借位；否则，该差值为 00。因此，上述表达式中的求和式就可以理解为借位发生的次数。这就得到了 Kummer 定理的文字表述。
